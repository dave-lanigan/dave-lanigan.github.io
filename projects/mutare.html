Helper/Classes Functions to make auto-making swapping of Cryptocurrencies possible.

The purpose for this project was to simplify and automate the process of swaping coins on certain decentralized, non-custodial exchanges, such as ShapeShift and Kyber Network. Thereby, allowing for algorithmic trading to take place. Code for this project can be found on my GitHub here.

Finding the Contract Address and ABI File.

Finding the contract address and ABI for the token can be tricky, usually you should go to Etherscan and search for the coin in the search bar. Then using the contract address you find search again for the ABI. However, for certain coins, such as Paxos, the ABI did not work. In this case, I was forced to go to Paxos GitHub and find it there in the paxos-contracts folder. This is probably the best way of getting it. Once the the contract address and ABI are acquired, you can add them to the contracts.json file.
