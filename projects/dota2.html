DOTA 2 Network Project

This master's degree class project involved creating a complex network graph from the OpenDota API data. Data from millions of matches was downloaded and used to make a network where nodes defined characters and edges between nodes were weighted based on win rates when characters were playing on the same team. If a threshold for a win rate is defined, the edges can be percolated to create networks with different numbers of edges. Code for this project can be found on my GitHub here.
