Crypto Data Trends Project

The purpose of this project is to download, consolidate, transform and prepare cryptocurrency price, volume and google trend data for analysis and exploration.

To accomplish this, data was acquired from (3) different sources. The first was coingecko API where a list of viable coin candidates were pulled, along with their fundamental information. The coins ranked in the top ~50 by market cap and supported on Binance were used in the final database. Binance was chosen for its ubiquity as a cryptocurrency exchange, strong API and fairly low trading fees. The historical data was then pulled from the Binance API using USDT (Tether) as the base currency. The google trends data was pulled from Google using a pytrends module. Then the data was consolidated into a postgres database organized by date. Code for this project can be found on my GitHub here.
