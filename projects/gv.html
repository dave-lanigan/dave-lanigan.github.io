This is a simulation of the Breadth First Search algorithm on an undirected graph.

First pick the size of the fully connected network represented by blocks.

Blocks represent node and adjacent blocks have edges between them - the most nodes a single node can be connected to is four.

The "start node" is selected for you at random. Select the "end node" and then begin to percolating the network by de-selecting the blocks you want to remove.
